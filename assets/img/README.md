# Dossier images

## profile.jpg

À remplacer par TA photo de profil :
- Format carré recommandé (500×500 px minimum)
- JPG ou PNG
- Photo terrain ou atelier (plus parlante qu'une photo studio)

## favicon.png

Petite icône qui apparaît dans l'onglet du navigateur.
- 32×32 ou 64×64 px
- PNG transparent
- Suggestion : tes initiales MLP sur fond simple

## Autres images

Pour les pages, dépose tes images ici et utilise dans le Markdown :
```markdown
![Légende]({{ '/assets/img/nom-image.jpg' | relative_url }})
```
